
# نظام حجز وإدارة العيادات الطبية - ClinicPro

## نظرة عامة
نظام متكامل لإدارة العيادات الطبية يهدف إلى تبسيط العمليات الإدارية وتحسين كفاءة الخدمة الطبية.

## المميزات الرئيسية
1. **إدارة المواعيد**: نظام حجز ذكي مع تجنب التعارض
2. **إدارة المرضى**: سجلات طبية إلكترونية كاملة
3. **الإدارة المالية**: فواتير، مدفوعات، ومصاريف
4. **التقارير**: إحصائيات وتحليلات مفصلة
5. **الأمان**: تشفير بيانات ونسخ احتياطي

## التثبيت
1. انسخ الملفات إلى سيرفر الويب
2. افتح `index.html` في المتصفح
3. اتبع خطوات الإعداد الأولي

## الاستخدام
- **المرضى**: يمكنهم حجز المواعيد عبر `booking.html`
- **الأطباء**: يمكنهم الدخول عبر `login.html` والوصول إلى `dashboard.html`
- **المسؤولين**: إدارة كاملة للنظام عبر `dashboard.html`
